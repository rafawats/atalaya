# Atalaya

Aplicación web local, multiplataforma y single-user para monitorear métricas del sistema (CPU, RAM, disco) del host donde se ejecuta.

**Estado:** en desarrollo (MVP). Ver [`IPP.md`](IPP.md) para el plan de iteraciones.
